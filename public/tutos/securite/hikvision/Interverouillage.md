# Lier unlecteur à une autre sortie

Pour faire cela il faut aller dans : controle d'accès/configuration de liaison.  
Il fauda ensuite créer un évènement du type **Evènement de lecture de carte** puis **authentification normale russie** puis dans l'onglet *Point de controle d'accès** choisir **Ouvrir**pour la ou les portes concernés.
